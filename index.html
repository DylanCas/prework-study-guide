HTML

  • The head element contains info about the webpage, for the browser.
  • The body element contains all the visible content on a webpage.
  • The Meta element item, 'UTF-8', is the most common for use in webpages.
  • Elements nested within other elements are called child elements, within the parent elements.
  • All content in a webpage is closed within "tags", usign the "<" and ">" symbols.
